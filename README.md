# 🎯 AI Placement Risk Predictor
--------------------------------------------------------------------------------------------
## 📌 Description
This project predicts student placement chances using Machine Learning and provides career suggestions. It also helps analyze potential risks in education loan repayment.
--------------------------------------------------------------------------------------------------
## 🚀 Features
- Placement Prediction
- Risk Analysis
- Internship Suggestions
- Probability Score
------------------------------------------------------------------------------------------------------------------------------
## 🧠 Machine Learning Model
- Algorithm Used: **Random Forest Classifier**
- The model is trained on student academic and skill-based data to predict placement outcomes.
---------------------------------------------------------------------------------------------------------------------
## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- ML: Scikit-learn
------------------------------------------------------------------------------------------------------------------
## ▶️ How to Run

### 1. Train Model
cd model  
python train_model.py  

### 2. Run Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### 3. Run Frontend
Open index.html in browser  
-------------------------------------------------------------------------------------------------------------------
## 💡 Output
- High / Moderate / Low placement chance
- Probability percentage
- Suggestions for improvement
